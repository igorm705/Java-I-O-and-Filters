This program includes files:
ResctrictedView: main class for starting the program.
CompositWebPageFilter: This class implements WebPageFilter and can includes two implementations of this: English filter and Javascript filter.
EnglishDocumentsFilter: this class implements WebPageFilter and blockes all sites if these languages not English
HTMLTagRemover : removes HTML tags from loaded page
ReadFromFile: open and read a file, and return the lines in the file as a list of Strings.
RejectJavaScriptFilter: filter for web-sites: blocks sites with java-script code inside
WebPageFilterFactory: the class for creating class of filter.

English filter uses checking for mostly common English words: it takes these words from external file
"99 common English words" and checks if minimum 5 words from this list appear in the text.

The filter for javascript gets whole text with HTML tags and seeks for string "< script >"inside

This project includes two configurations: each one for one specific filter: first with name "English Filter"
with arguments "-english https://fr.wikipedia.org/wiki/Wikip%C3%A9dia:Accueil_principal" and second "Javascript Filter" with arguments
"-javascript http://www.hac.ac.il"
